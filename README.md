# Angular UI Component Library

Proyecto desarrollado como parte de una prueba técnica, donde se creó una librería reutilizable en Angular y una aplicación de demostración para consumir el componente.

## 🚀 Tecnologías utilizadas

- Angular
- TypeScript
- HTML
- CSS
- Node.js
- npm
- Git
- GitHub

## ✨ Funcionalidades

- Creación de una librería Angular.
- Desarrollo de un componente Button reutilizable.
- Integración del componente en una aplicación demo.
- Configuración con Angular CLI.
- Control de versiones con Git.

## 📚 Lo que aprendí

- Configuración de un entorno Angular.
- Creación de librerías reutilizables.
- Uso de Angular CLI.
- Gestión de dependencias con npm.
- Publicación de proyectos en GitHub.
